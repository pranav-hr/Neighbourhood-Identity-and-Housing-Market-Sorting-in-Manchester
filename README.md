# Neighbourhood Identity and Housing Market Sorting in Manchester

## Overview

This repository contains the code and documentation for an MSc dissertation examining whether Wikipedia-described neighbourhood identity explains where new build and resale property transactions occur in Manchester, using a discrete choice framework.

**Research Question:** Do new build transactions sort into neighbourhoods based on Wikipedia-described identity more strongly than resale transactions, and does this hold after controlling for observable neighbourhood characteristics?

**Key Finding:** Wikipedia sentence transformer embeddings achieve better model fit (McFadden ρ² = 0.132) than the full suite of official statistics — price, deprivation, crime and income — (ρ² = 0.123) in explaining spatial transaction patterns, and this advantage is concentrated in the new build market.
